📍 모바일(ios) 위치 설정 확인

  1. 설정 → Safari → 위치 → 묻기 또는 허용
  2. 설정 → 개인정보 보호 및 보안 → 위치 서비스 → Safari → 묻기 또는 앱을 사용하는 동안 허용
